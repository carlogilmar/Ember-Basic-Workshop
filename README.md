# Ember Basic Workshop

## Installation

* `npm install`

## Running / Development

* `ember serve`

### Running Tests

* `ember test`

### Building

* `ember build` (development)
* `ember build --environment production` (production)

## Workshop

1. Create a simple application
2. Simple Templates with Handlebars
3. Router and routes
4. Actions
5. Models
6. Components
7. Computed Properties
8. Controllers
9. Deploy
